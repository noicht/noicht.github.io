date = 2026-06-22
+++

I didn’t want to learn Java by building another to-do app.

I wanted a project that forced me to think about:
- interfaces and abstractions
- long-lived architecture decisions
- package structure
- command-line workflows
- testability
- code that can grow without collapsing under its own weight

That project is Sentinel.
